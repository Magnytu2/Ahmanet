# Changelog du Template Ahmanet

## Version 1.0.0 (16 août 2025)

### Ajouts
- Création initiale du template enfant Ahmanet basé sur Atum
- Intégration des couleurs MDBootstrap avec variables CSS personnalisables
- Ajout d'un onglet de configuration "Couleurs de Ahmanet" dans les paramètres du template
- Création du fichier template-colors.php pour la génération dynamique des variables CSS
- Création du fichier mdbootstrap-components.css pour les styles des composants UI
- Création de logos personnalisés (login, principal, petit)
- Ajout du fichier logos.css pour la personnalisation de l'affichage des logos
- Support multilingue avec fichiers de traduction en français et anglais
- Remplacement des icônes Font Awesome par des icônes Material Design Two Tone
- Documentation complète du template

### Modifications
- Personnalisation du bouton de profil utilisateur avec style MDBootstrap
- Augmentation de la taille de l'icône d'accueil
- Optimisation des styles pour une meilleure compatibilité mobile

### Corrections
- Correction de la propriété font-feature-settings pour la compatibilité des navigateurs

## Prochaines évolutions prévues
- Amélioration de l'accessibilité
- Optimisation des performances
- Ajout de nouveaux composants UI
- Extension du support multilingue
